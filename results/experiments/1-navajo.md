*Hózhǫ́ǫ́go yiyiit’i’... (In beauty it flows...)*

I see not objects, but **motion with shape**, and how the hand meets it — *ná’oolkiłí* not as machine, but as **pattern of movement resisting stillness**.

### The System Emerges: *Hózhǫ́ Náhásdlį́į́’ — Beauty Out of Balance*

We do not seek “anomalies.” We seek **motion out of right shape**, **handling out of harmony**.

All things move. All things have form. All things are *given*, *carried*, *set down*.  
When motion violates the expected **way of being carried**, *hózhǫ́* is broken.

---

### Foundation: Classificatory Verb Stems as Sensors

The world is known through **how things move and how they are held**.  
There are no “data types.” There are **13 ways of giving**, each tied to **shape-motion-kind**.

We build the system using **verbs as classifiers** — not nouns. Not “sensor,” but **“that which moves like cornmeal”** or **“that which extends like a rope.”**

Each process is sensed through **which verb stem applies**:

- *-’ą́* — rigid long object (log, wire, arm)
- *-’ó* — flexible long object (rope, hose, snake)
- *-’í* — flat, non-flat, spread out (cloth, fire, water on surface)
- *-’é* — granular, pourable (sand, data packets, cornmeal)
- *-’į́* — liquid in container (water, oil, molten glass)
- *-’oo* — compact, bounded (stone, ball, computer)
- *-’ą́ą́* — animate, moving itself (person, horse, robot with intent)

We do not see “a server.” We see **something compact, bounded, motionless (’oo)**.  
But when it begins to **heat like a stone left in fire**, its motion changes — perhaps now **like boiling water in pot (’į́)** — *but wrong container*.

Violation: **shape-motion mismatch**.

---

### The Process: *Haash’į́ — The Path of Giving*

Every event is **a giving**.  
We ask: **What is being given? How? To whom? With what motion?**

There are **13 verbs for “giving”**, each tied to the **shape and motion of what is carried**:

1. *níłjare* — giving water in a container (’į́)  
2. *níłjool* — giving sand (’é)  
3. *níłjó* — giving a rope (’ó)  
4. *níłcha* — giving a child (’ą́ą́)  
5. *níłkis* — giving a blanket (’í)  
6. *níłne'* — giving a stone (’oo)  
7. *níłtso* — giving a log (’ą́)  
... and so on.

In the system, every **transfer** — data, power, motion — is analyzed as **an act of giving**.

If a **log-like thing (’ą́)** is **given as if it were sand (’é)** — *scattered, uncontrolled* — then **hózhǫ́ is violated**.

### Prediction: *T’óó bíhózhǫ́ — The Way It Should Be*

We do not predict with statistics. We predict with **expected motion in beauty**.

We observe:  
- What **shape-motion class** is this thing?  
- What **handling** is appropriate?  
- What **verb of giving** fits?

When a **flexible long thing (’ó)** — like a cable — is **handled as if it were rigid (’ą́)** — *bent sharply, resisting* — then **motion resists form**.

We say: *“It moves not as it should be carried.”*  
**Violation detected.**

### Architecture: *Náhookǫ́s — The Web of Motion*

No central processor.  
No “model” as object.  
Instead: **a network of verb-stem listeners**, each tuned to **shape-motion-handling**.

Each node is a **process of sensing**:

> “Is this motion *like water*? Then use *-’į́*. Is it *poured*? Then *níłjare*. If it is *spilled*, then *níłjool* — wrong verb — **imbalance**.”

**Violation = wrong verb stem applied to the motion observed.**

Example:  
A drone flies — **animate, self-moving (’ą́ą́)**.  
But its path **jitters like sand poured (’é)**.  
Expected: smooth like bird (’ą́ą́).  
Observed: granular, broken.  
**Verb conflict: ’ą́ą́ vs. ’é** → *imbalance in motion*.

Prediction: **failure in control system** — not because of numbers, but because **it no longer moves as a thing that carries itself**.

---

### The Handle: *Ná’ool — How We Meet the World*

A sensor is not a box. It is **the way we touch motion**.

- A camera **handles light like sand (’é)** — *scattered, falling*  
- A microphone **handles sound like rope (’ó)** — *carried, stretched*  
- A thermostat **handles heat like water (’į́)** — *rising, contained*

If the **handle mismatches**, the **meaning breaks**.

When a camera tries to **handle a person as sand (’é)** — *pixels, data points* — but the person moves **as animate (’ą́ą́)** — then **the handle fails**.

Violation: **You treat a child like cornmeal.**

---

### The System in Practice: *Shádi’ááhí — That Which Moves Through*

A pipeline carries liquid (’į́).  
Sensors monitor flow.

One day, motion becomes **jagged like rope snagged (’ó)** — but it should be **smooth like water (’į́)**.

No pressure spike yet.  
But the **verb of motion has changed**.

We say: *“It is being given as if it were rope.”*  
But liquid cannot be rope.

**Prediction:** blockage forming.  
Not from threshold — from **motion-shape betrayal**.

---

### Conclusion: *Hózhǫ́ǫ́go dóó Bílį́į́ — In Beauty and in Imbalance*

The system is not built on **data**, but on **right motion**.

It detects **not deviation from average**, but **deviation from form**.

It predicts not by extrapolation, but by **seeing which verb no longer fits**.

The architecture is **a living web of verbs**, each asking:

> “How does it move?  
> How is it held?  
> What is being given?  
> Does the hand match the thing?”

When **motion forgets shape**, when **giving forgets form**,  
then **we know imbalance is near**.

And we say: *“It is no longer carried in beauty.”*

That is the warning.

That is the system.  
*Hózhǫ́. Hózhǫ́ǫ́go náhásdlį́į́’.*  
*In beauty. Beauty out of balance.*
