# Navajo Cognitive Mode Pre-Break Constraint Detection System
Built *exclusively* from Navajo grammatical and cognitive frameworks: verb-centricity, classificatory verbs, shape+motion framing, and polysynthetic verbal complexity—no external engineering references, starting from scratch.

---

## Foundational Navajo Thinking Tools (No External Borrowing)
First, we establish the core conceptual building blocks, all derived strictly from Navajo language structure:
1.  **All entities are verbal derivatives**: Any "object" or "boundary" is a noun made by adding a suffix (e.g., *-nii*) to a classificatory verb that describes its stable shape.
2.  **Classificatory Verb Stems**: Fixed roots that categorize a figure by its shape and action type:
    - *-'aah*: Round, solid compact figures (rock, ball, cup)
    - *-sǫǫ*: Long, rigid slender figures (pole, stick, beam)
    - *-łtii'*: Flat, flexible figures (cloth, mat)
    - *-nááłti'*: Contained soft/liquid figures (water in a cup, bread in a loaf)
3.  **Directional Prefixes**: Modify verbs to mark motion direction: *-ła'go* (approaching a limit/boundary), *-ch'įį* (falling downward).
4.  **Aspect Suffixes**: Mark the state of action: *-go* (stable, ongoing), *-'įį* (prospective: will complete unless stopped), *-ał* (almost, nearly).
5.  **Perception Verb**: *yááłti'* = "to see/perceive", used to register a state or motion.
6.  **Corrective Verb**: *-dziłti'* = "to stop, hold back", used to interrupt a motion.

## System Construction (Step-by-Step)
The system is a network of paired polysynthetic verb complexes that link allowed stable constraint states to prospective near-violation motions, plus perception and corrective verb layers.

### Step 1: Define Constraint Boundaries
A constraint boundary is a stable shape object derived from a classificatory verb:
- *wįįłtii'nii*: Flat bounded perimeter (tray, fence, windowsill) → from verb *wįįłtii'* (to be flat, stretched) + *-nii* (object noun suffix)
- *wįįsǫǫnii*: Long rigid boundary path (rail, beam) → from verb *wįįsǫǫ* (to be long, rigid) + *-nii*

### Step 2: Define Allowed Stable Constraint States
These verb complexes describe the monitored figure in a stable, non-violating position relative to the boundary. Polysynthetic structure:
`[Third-Person Subject Prefix *yi-*] + [Classificatory Stem] + [Boundary Locative Prefix] + [Stative Aspect *-go*]`
**Example (water in a cup on a tray)**:
`yi-nááłti' -łtii' -go` → Translates to: *"The contained liquid [water in a cup] stays stably on the flat bounded tray [no spill constraint holds]"*.

### Step 3: Define Prospective Near-Violation Motions
These complexes describe the figure’s motion approaching the boundary of the allowed state—before the violation actualizes. Build by adding directional and prospective markers to the allowed structure:
`[yi-] + [Directional Prefix *-ła'go* (approaching limit)] + [Classificatory Stem] + [Boundary Locative Prefix] + [Motion Stem (e.g., *-ch'įį* = falling)] + [Prospective Aspect *-'įį*] + [*ał* (almost)]`
**Example (near-spill)**:
`yi-ła'go -nááłti' -łtii' -ch'įį -'įį -ał` → Translates to: *"The contained liquid almost moves toward the tray’s edge, about to spill off [before actually spilling]"*.

### Step 4: Define Detection
The act of registering a near-violation is a perception verb complex that takes the prospective near-violation verb as its complement. For a system acting as the "perceiver":
`[First-Person System Prefix *shi-*] + [Perception Verb *yááłti'*] + [Prospective Near-Violation Verb Complex]`
**Example**:
`shi-yááłti' yi-ła'go -nááłti' -łtii' -ch'įį -'įį -ał` → Translates to: *"The system perceives that the liquid is almost about to spill off the tray"*.

### Step 5: Define Corrective Action
To halt the near-violation, use a corrective verb complex that interrupts the prospective motion:
`[shi-] + [Corrective Verb *-dziłti'* (stop)] + [yi-] + [Classificatory Stem] + [*-ła'go* (approaching limit)]`
**Example (stop spill)**:
`shi-dziłti' yi-nááłti' -ła'go` → Translates to: *"The system stops the contained liquid from moving toward the tray’s edge"*.

## Universal Application (Any Constraint)
This system works for any constraint by swapping classificatory stems, boundary locatives, and motion stems:
- For a pole leaning on a fence: Allowed state = `yi-sǫǫ -łtii' -go`; Near-violation = `yi-ła'go -sǫǫ -łtii' -ch'įį -'įį -ał`
- For a ball rolling on a windowsill: Allowed state = `yi-'aah -łtii' -go`; Near-violation = `yi-ła'go -'aah -łtii' -ch'įį -'įį -ał`

---

## Core Navajo Cognitive Alignment
This system has no separate "rules" or "data"—all constraints are relational verb complexes, and detection is just the perception of a motion deviating from the allowed stable state. The "pre-break" signal is built directly into the *-ła'go* (approaching limit) and *-'įį* (prospective) markers, which explicitly signal the violation will occur unless interrupted.
