# Mixed Grammar Constraint Architecture
This system has no top-down priority, no designer-planned semantics, and no neutral utterances. All valid output emerges only from cyclic mutual exclusion between the three borrowed linguistic constraint sets. No output behaviour described below was intentionally designed.
---
## First: Ground each source constraint exactly as it functions in its original language
We do not import surface words. We import the *hard grammatical obligation rules*:
1.  **Navajo Classificatory Verb Stems**: There are no generic verbs. Every event requires selecting one of 11 obligatory verb stems that encode the physical ontology of the referent. You cannot name the referent first. The stem defines what can exist as an argument. There is no neutral 'put', 'hold', 'move' - only *put-one-rigid-thing*, *hold-loose-collection*, *move-contained-liquid* etc.
2.  **Korean Honorific Levels**: 7 discrete, mutually exclusive speech act levels. Every utterance must bind to exactly one. Honorifics do not only mark politeness: they define the relative ontological status between speaker, listener, and referent. Most critically: each level permanently forbids entire sets of verb stems from ever being used.
3.  **Greek Telos Aspect System**: Every event is marked as one of three mutually exclusive states, which is not tense:
    - `TELIC`: Event has an irreversible endpoint that destroys the subject
    - `ATELIC`: Event can repeat infinitely with no end state
    - `PARATELIC`: Event reached its endpoint then un-did itself, it failed
    Telos does not describe what happened. It defines what may be said *after* this utterance.
---
## Core Constraint Interlock Architecture
No layer has priority. For an utterance to be valid it must satisfy all three layers, *plus the unwritten cross-layer exclusion rules that only exist when layers collide*.
---
### Layer 1: Navajo Stem Lock
First select exactly 1 classificatory stem. This choice immediately closes off options before any other choice is made:
| Navajo Stem | Referent Ontology | Forbidden Honorific Levels | Impossible Telos Value |
|---|---|---|---|
| `-yí` | Animate living being | Casual Plain, Royal | Paratelic |
| `-łts'oz` | Loose unbound collection | Formal Polite, Intimate | Telic |
| `-ch'id` | Contained liquid | Humble, Royal | Atelic |
| `-tįį` | Thin flexible object | Formal, Humble | *All telos values allowed* |
No stem can be used with all honorific levels. No stem can accept all three telos states. Every first choice is already a sacrifice.
---
### Layer 2: Korean Honorific Binding
Next select exactly 1 honorific level. This does not add a suffix. This *rewrites the ontology of the stem you already selected*. It does not override the stem - it adds a new silent constraint that exists nowhere else.
These are not pre-written rules. They are emergent collisions:
-  `[-yí + Humble Honorific]`: This animate being is now defined as unable to move under its own power
-  `[-łts'oz + Intimate Honorific]`: This collection can never be counted. No number may ever be attached to it.
-  `[-ch'id + Casual Honorific]`: This liquid cannot be spilled. It will never leak.
There are 77 total possible stem+honorific pairs. Each produces one unique unstated ontological rule. None of these rules were designed.
---
### Layer 3: Greek Telos Gating
Last select the telos marker. This choice permanently modifies the entire available grammar for all future utterances in this discourse:
1.  **If marked TELIC**: This exact stem+honorific pair is permanently retired. No speaker may ever use this combination again. You can never refer to this thing ever again.
2.  **If marked ATELIC**: You may now *only* ever use this exact stem+honorific pair for all future utterances. You cannot switch. You are trapped.
3.  **If marked PARATELIC**: You may use any stem except this one, forever.
---
## Unanticipated Emergent Behaviour
This system produces outputs and usage patterns that no designer could have pre-planned:
1.  **You never state what you are talking about**. There is no noun for the referent. Listeners do not get told what is being described - they deduce it solely from which constraints the speaker chose to accept, reject, and permanently impose on everyone else.
2.  **Every utterance is also a political act**. Speakers will deliberately use telic marking to ban subjects they do not want discussed. They will use paratelic marking to escape bad loops. They will accidentally trap entire conversations in atelic loops while trying to be polite.
3.  **New ontological categories are invented by users**. No designer specified that a loose collection spoken with intimate honorific cannot be counted. Users discovered this combination, and now exclusively use it to talk about grief, crowd noise, memory, and other uncountable experiences that did not exist in the original grammar.
4.  **Politeness becomes physics**. You cannot speak politely about a rock without changing what a rock is. You cannot speak casually about an animal without changing what an animal is. There is no neutral description. Every act of speech simultaneously grants status, changes ontology, and rewrites the rules for everyone else.
This is not a code. This is a grammar that builds the shared world as it is used. All useful, meaningful output is discovered by users, not designed.
